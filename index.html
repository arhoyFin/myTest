To initialize folder a a git repo, open up GIT bash and type:
git init

By default, it won't show up it is a hidden file. You can change that by going to Control Panel > Folder Options >View . Under Advanced settings , there should be an item called Hidden files and folders , with an option to Show hidden files, folders, and drives .

You should now see a folder .git which means we can use git command as the folder has been initialized as a Git Repository.

Now add your name + email address to GIT by doing git config --global user.name 'aquasar' user.email

List of git commands:

  • git init
    • initialize folder to set up git rep and use git commands
  • git status
    • this means to put changes made to files in the staging area.
  • git commit
  • git commit -m 'comments here to the changes made'
  • git rm --cached <file name>
    • remove the file
  • *.html
    • add all htm files to staging area.
  • git add .
    • adds everything to the staging area.
